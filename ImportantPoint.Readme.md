### Spring Framework有哪些核心模块
```
spring-core: Spring基础API模块 , 如资源管理,泛型处理;  
spring-beans: Spring Bean 相关, 如依赖查找,依赖注入;  
spring-aop: Spring AOP 处理, 如动态代理, AOP字节码提升;  
spring-context: 事件驱动、注解驱动，模块驱动等； 
spring-expression: Spring 表达式语言模块;  
```
